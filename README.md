# Portfolio Website

Portfolio website for Full Stack Developer Malene Thisgaard, built with Nuxt 4, featuring projects, skills, and a responsive design.

## Features

- 🌐 **Multilingual** - English and Danish support
- 🌙 **Dark Mode** - Dark theme by default with color mode support
- 📱 **Responsive** - Mobile-first design with Tailwind CSS
- ⚡ **Fast** - Static site generation with Nuxt 3
- 🎨 **Modern UI** - Clean and minimalist design

## Tech Stack

- [Nuxt 4](https://nuxt.com/) - Vue.js framework
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [Vue I18n](https://vue-i18n.intlify.dev/) - Internationalization
- [Phosphor Icons](https://phosphoricons.com/) - Icon library

## Getting Started

### Prerequisites

- Node.js 18+ (recommended: 22+)
- npm

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev
```

The app will be available at `http://localhost:3000`

### Build

```bash
# Build for production
npm run build

# Generate static site
npm run generate

# Preview production build
npm run preview
```

## Project Structure

```
├── assets/          # Images and CSS
├── components/      # Vue components
├── constants/       # Data and constants
├── i18n/           # Translation files
├── layouts/        # Layout components
├── pages/          # Route pages
└── public/         # Static assets
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run generate` - Generate static site
- `npm run preview` - Preview production build
- `npm run lint` - Run linters
- `npm run lintfix` - Fix linting issues
